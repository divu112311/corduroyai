# TradeAI Python

This repository contains Python code for TradeAI-related projects.

## Structure
- `tradeingestion/` – data ingestion scripts
- `tradeai/` – core logic and utilities
- `Scripts/` – helper scripts
- `corduroyai/` – experimental or supporting code
